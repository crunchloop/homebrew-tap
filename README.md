# Crunchloop Homebrew Tap

Homebrew casks for Crunchloop's tools. **Managed automatically by
[GoReleaser](https://goreleaser.com)** on each release — don't edit `Casks/` by hand.

## Usage

```sh
brew install crunchloop/tap/<tool>
```

For example:

```sh
brew install crunchloop/tap/medea
```

`brew install crunchloop/tap/<tool>` auto-taps this repo, so a separate
`brew tap crunchloop/tap` isn't required.
